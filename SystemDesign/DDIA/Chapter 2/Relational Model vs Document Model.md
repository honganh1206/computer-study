# Relational Model vs Document Model

> [!summary]
> - Relational mode was born to solve many-to-many relationships that hierarchical model could not solve. 
> - NoSQL data stores have two main directions: Document databases (Self-contained documents) and Graph databases (Anything is potentially related to everything).
> - Document and graph databases do not enforce schema-on-write.
