A **transaction** in programming context refers to a *group* of reads and writes that forms a logical unit.

The Online Transaction Processing (OLTP) basic access pattern is similar to processing business transactions

Analytics: Scan over rows, read a few columns per record, and calculate aggregate statistics. Usually called Online Analytics Processing (OLAP).

A trend to run OLAP on a separate database (data warehouse)
